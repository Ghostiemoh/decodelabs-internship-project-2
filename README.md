# E-Commerce Data Analytics & Executive Dashboard (Week 2)

This repository contains the dataset and documentation for the Week 2 data analysis project. The project involves generating descriptive statistics, monthly/yearly trends, channel metrics, and coupon performance insights for an e-commerce platform.

## Repository Contents

The following files are included in this project:

| File Name | Format | Description |
| :--- | :---: | :--- |
| `Dataset for Data Analytics P2.xlsx` | Excel Workbook | The dataset containing the main transaction data and a new, styled `Executive Summary` dashboard tab. |
| `Data analytics P2.pdf` | PDF Document | The original project guidelines and instruction manual. |

## Dashboard & Sheet Highlights

The `Executive Summary` sheet was added to the front of `Dataset for Data Analytics P2.xlsx`. To ensure the dashboard remains dynamic, it was built entirely with Excel formulas rather than static numbers.

* **Core Performance Metrics**: Uses formulas like `AVERAGE`, `MEDIAN`, `MIN`, `MAX`, and `STDEV` to analyze transaction volumes, cart sizes, and revenue.
* **Fulfillment status funnel**: Analyzes the order funnel using `COUNTIF` to show how different states perform.
* **Product Performance Profile**: Groups and sums quantity, revenue, and average unit price per product (Chairs, Laptops, Phones, etc.) using `SUMIF` and `AVERAGEIF`.
* **Acquisition Channels**: Breaks down total orders and revenue generated by Instagram, Email, Google, Facebook, and Referrals.
* **Coupon return rate correlation**: Calculates the order count, revenue, and post-purchase return rate for each coupon code to check return behaviors.
* **Yearly Trends**: Offers a year-over-year H1 (First Half) revenue breakdown to track sales contraction without seasonal bias.

## Key Observations

1. **Fulfillment Funnel Leakage**: A significant 41.4% of all orders are lost to Cancellations (20.8%) and Returns (20.6%). Only 19.3% of orders result in completed deliveries.
2. **Sales Contraction**: The first-half (H1) revenue shows a compounding contraction of approximately 10% year-over-year ($286.5k in H1 2023 vs $231.9k in H1 2025).
3. **Coupon Retention Impact**: Orders placed without a coupon code suffer from the highest return rate (24.6%), while using a `SAVE10` coupon reduces returns to 16.4%.
